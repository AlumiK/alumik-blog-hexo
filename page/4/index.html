Concatenation of Files with Same Codecs

There are two methods within FFmpeg that can be used to concatenate files of the same type: the concat demuxer & the concat protocol

The demuxer is more flexible ‚Äì it requires the same codecs, but different container formats can be used; and it can be used with any container formats, while the concat protocol only works with a select few containers.
