---
title: 设置默认采用 root 用户登录 WSL
date: 2019-06-24 15:40:21
categories: WSL
tags:
---
在命令行（管理员）中输入以下命令

```
<WSL 发行版代号> config --default-user root
```

目前可用的发行版不完全列表如下

| 发行版代号 | 发行版名称 |
| - | - |
| `ubuntu` | Ubuntu |
| `ubuntu1804` | Ubuntu 18.04 |
| `opensuse-42` | openSUSE Leap 42 |
| `sles-12` | SUSE Linux Enterprise Server 12 |
| `debian` | Debian GNU/Linux |
| `kali` | Kali Linux |
